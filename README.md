# Java
scripts feitos enquanto estudo Java

# Classes
- [scan](Class/scan.java) - classe com metodos para a entrada de dados
    - scan_int - lê numeros inteiros da entrada padrão
    - scan_float - lê numeros decimais da entrada padrão
    - scan_double - lê numeros com casas decimais de dupla precisão da entrada padrão
    - scan_char - lê caracteres da entrada padrão
    - scan_string - lê strings da entrada padrão
- [math](Class/math.java) - classe com metodos para operações matematicas
    - sum - lê dois numeros inteiros da entrada padrão e retorna a soma deles
    - multiplication - lê dois numeros inteiros da entrada padrão e retorna a multiplicação deles
    - subtration - lê dois numeros inteiros da entrada padrão e retorna a subtração deles
    - reason - lê dois numeros inteiros da entrada padrão e retorna o quociente deles
- [string](Class/string.java) - classe com metodos para manipulações de strings
    - palindromo - metodo para verificar se uma string é um palindromo
    - scannerEOF - metodo para ler um string até o EOF - End Of File
    - anagramas - metodo para verificar se duas palavras são anagramas
